This is UCLA Meme Page The Game. It requires QT to run. 

Optimal Kit nd Compiler to run our program:
QT Version: Qt 5.12.5 clang 64 bit
Kit: Desktop Qt 5.12.5 clang 64bit
CMake: 3.16.0
Compiler: Clang(C++, x86 64 bit in /usr/bin)

First it will open an introduction screen, where options to change difficulty level, play with or without mute, start the game, and to see instructions will be displayed. These options will be accessible by clicking. Once the user presses play, they can use their keyboard to navigate. 

W=Up
A=Left
S=Down
D=Right
Space=Shoot


The objective is to protect Gene Block, which can be done by shooting the zombies. If a zombie collides with Gene Block, his health decreases and the game ends.
